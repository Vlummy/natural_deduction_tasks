To run this app:

Requirements:
1. NodeJS installed

This is a React app served using ExpressJS server.

To run this app write the following in the console.

1. open console in root folder of this project.
2. > npm install
3. > npm run build
4. > npm run start

In your browser open URL:
http://localhost:8080/

Another option is to test the app using the provided URL below:
https://nd-learning.herokuapp.com/
